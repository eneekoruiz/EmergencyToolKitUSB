# Inventario Hardware del Arsenal Digital

Un Arsenal Digital perfecto no sirve de nada si llegas al centro de datos del cliente y te falta el cable adecuado. Revisa esta lista antes de salir.

## Almacenamiento & Medios
- [ ] USB Principal: Rescate & Ventoy (mínimo 64GB, USB 3.2+).
- [ ] USB Secundario: Privacidad (Tails) / Operativo Limpio.
- [ ] USB Terciario: Auditoría Ofensiva (Kali Linux).
- [ ] Disco Duro Externo (SSD recomendado, mínimo 1TB) - Para volcados forenses de RAM, imágenes completas (dd, Clonezilla) y copias de seguridad de perfiles.

## Conectividad y Redes
- [ ] Cable Ethernet RJ-45 (x2).
- [ ] Cable Ethernet Cruzado (para conexión directa PC a PC sin switch).
- [ ] Adaptador USB a Ethernet (Gigabit) - Vital porque muchos portátiles modernos no tienen puerto RJ-45.
- [ ] Adaptador Wi-Fi USB (compatible con Monitor Mode / Inyección de paquetes) - Ej. adaptadores con chipsets Alfa/Atheros.
- [ ] (Opcional) Mini Switch de 5 puertos - Para intervenciones donde necesitas conectar tu equipo y hacer port mirroring (con un tap de red).

## Adaptadores de Almacenamiento
- [ ] Adaptador NVMe (M.2) a USB-C/A.
- [ ] Adaptador SATA (2.5") a USB.
- [ ] (Opcional) Adaptador para discos viejos IDE a USB (solo si trabajas con legacy).

## Herramientas Físicas Básicas
- [ ] Destornillador multipunta (con cabezales Torx, Philips pequeños) - Para abrir portátiles y extraer el disco si el equipo no enciende en absoluto.
- [ ] Linterna pequeña o frontal.
- [ ] Bridas / Velcro para ordenar el caos en los racks.
- [ ] Pinzas extractoras.

## Miscelánea y Supervivencia
- [ ] Batería externa (Powerbank) para cargar móviles o alimentar pequeños aparatos.
- [ ] Adaptador universal de corriente.
- [ ] Bloc de notas físico de papel y bolígrafo (a veces es lo único que no falla).
