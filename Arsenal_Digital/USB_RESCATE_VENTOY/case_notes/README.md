# Bitacoras de caso

Crea una bitacora por intervencion:

```powershell
..\..\scripts\New-RescueCaseLog.ps1 -CaseName CLIENTE-EQUIPO-INCIDENTE
```

Registra siempre:

- Fecha y zona horaria.
- Equipo, usuario responsable y autorizacion.
- Discos detectados y numeros de serie.
- Acciones ejecutadas.
- Hashes de imagenes generadas.
